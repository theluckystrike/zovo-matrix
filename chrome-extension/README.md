# Privacy Shield — Chrome Extension

A lightweight Chrome extension that blocks trackers, analytics, and other privacy-invasive requests directly in your browser.

## What It Does

Privacy Shield intercepts and blocks network requests to known tracking and analytics domains before they leave your machine. No data is sent, no profiles are built, no cookies are dropped.

## Installation

1. Open Chrome and navigate to `chrome://extensions`.
2. Enable **Developer mode** (toggle in the top-right corner).
3. Click **Load unpacked**.
4. Select the `chrome-extension/` directory from this repository.
5. The extension icon should appear in your toolbar.

## What It Blocks

- Third-party analytics and tracking scripts
- Ad network requests
- Known fingerprinting endpoints
- Telemetry and data-collection beacons

## Privacy Policy

This extension does not collect, store, or transmit any user data. All blocking happens locally on your device.
